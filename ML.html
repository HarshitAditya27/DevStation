What is machine learning?

Machine learning (ML) is an artificial intelligence (AI) process that allows software applications to predict outcomes more accurately without explicit programming. Machine learning algorithms use historical data as input to predict new outcomes. Machine learning algorithms build a mathematical model based on sample data, known as "training data", in order to make predictions or decisions without being explicitly programmed to do so. learn more

Subcategories of Machine Learning

Machine Learning Cateogories into Supervised, Unsupervised, Reinforcement and Clustering learning, and more

Supervised learning algorithms can be divided into two categories:

  1. Regression
  2. Classification

Unsupervised learning can be divided into two categories:

  1. Dimensionality Reduction
  2. Clustering
