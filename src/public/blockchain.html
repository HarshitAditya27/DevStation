What is Blockchain?

Blockchain is a process of storing information in a way that makes it difficult or impossible to change, hack, or cheat the system.A blockchain is essentially a digital ledger of transactions that is duplicated and distributed across the entire network of computer systems on the blockchain. Each block in the chain contains a number of transactions, and every time a new transaction occurs on the blockchain, a record of that transaction is added to every participantâ€™s ledger. The decentralized database managed by multiple participants is known as Distributed Ledger Technology (DLT).

Roadmap for learning Blockchain

  • Learn Data Structures : Data Structures such as s linked lists, binary trees, heap, hashing, graph plays an important role in understanding blockchain.

  • Learn Cryptography: Cryptography is the study of secure communications techniques that allow only the sender and intended recipient of a message to view its contents. Cryptography plays an important role in the decentralization of blockchain

  • Learn Smart Contract and Solidity: A smart contract is simply a program that runs on the Ethereum blockchain. It's a collection of code (its functions) and data (its state) that resides at a specific address on the Ethereum blockchain. Solidity is a object oriented programming language which used for creating smart contacts
  • Learn about various blockchain technologies such Ethereum and its application in various fields.
